# 一、概述：什么是「崩老头」

**「崩老头」** 是网络黑话，指年轻女性（或伪装成女性的男性）以情感陪伴、暧昧互动为手段，向有经济基础的中年男性小额高频索取钱财的行为。

| 黑话 | 真实含义 |
|------|---------|
| 崩 | 北方方言，骗、榨取、变现 |
| 老头 | 并非老人，多指 30-50 岁、有稳定收入、情感空虚的中年男性 |
| 精神小妹 | 从事该行为的年轻女性从业者 |
| 养肥 | 让目标产生深度情感依赖后再深度收割 |
| 三暖一启 | 三天铺垫情感、第四天开始首次索财 |
| 三聊一索 | 后续每聊三天，索一次 |
| 代崩 | 接手「养肥」账号继续收割的下游团队 |

## 这件事的全貌（一张图看懂）

```
精准筛选目标（30-50岁情感空虚男性）
        ↓
打造温柔人设（27岁护士、AI 照片、视频素材）
        ↓
3-7 天情感铺垫（早安晚安、朋友圈互动、深夜陪聊）
        ↓
小额高频收割（奶茶 20 元、打车 30 元、咖啡 20 元……）
        ↓
话术升级（关联上次正反馈、节点收割、生日借口）
        ↓
进阶手段（外卖代付链接、视频分级收费、礼物施压）
        ↓
单笔永远小到不报警
累计金额可超 8 万元
        ↓
受害者因金额小 / 怕丢人 / 不确定性质 → 沉默
        ↓
灰产持续运转
```

> 💡 **关键洞察**：崩老头之所以能运转，不是因为它技术多高明，而是因为**它把每一笔都设计成「不像诈骗」**。20 块、30 块、52 块——看起来都是「自愿的小礼物」。直到受害者某天突然反应过来，账户已经被掏空了。

# 二、DAY -3：识别篇 —— 你正在被崩吗？

**这一步不需要专业知识，只需要诚实。**

回答以下问题。如果有任意 **3 个**回答「是」，请直接跳到第六章报警篇。

## A. 关于她

- [ ] 她**永远不接视频通话**，或者视频通话时**只露侧脸/逆光/光线极暗**
- [ ] 她的朋友圈**永远只有几个月内的内容**，没有更早的
- [ ] 她的照片大多是**背影、侧脸、戴帽子、戴口罩**——从来看不清全脸
- [ ] 你提过视频/见面，得到的回复永远是「下次吧」「过几天」「最近不方便」
- [ ] 她的工作、地址、生活细节**前后不一致**，被你指出时她会说「你记错了吧」「怎么这么不信我」
- [ ] 她从不主动发长段语音，**语音消息极少**，问起来她说「嗓子不舒服」

## B. 关于钱

- [ ] 她**经常哭穷**——工资低、房租贵、家里有事
- [ ] 第一次要钱是**很小一笔**（奶茶 20、打车 30、咖啡 25）
- [ ] 第一次要钱时她说「算了不喝了」，**让你更想给**
- [ ] 每次要钱都有**具体的场景**（加班累了、夜班、想妈妈、过生日）
- [ ] 索财频率越来越密，金额越来越大（20 → 50 → 100 → 200）
- [ ] 她收到钱后的反应**强烈而具体**：「啊啊啊啊谢谢哥哥」「你是第一个给我买 X 的人」
- [ ] 你不主动转账时，她会**情绪化**：「你果然不爱我」「我以为你懂我」

## C. 关于关系

- [ ] 你从没**真正见过她的家人朋友**
- [ ] 她对你的称呼**亲密度上升很快**（几天内从「你」变成「哥哥」）
- [ ] 她经常**凌晨主动找你聊天**，但你白天找她经常「在忙」
- [ ] 她极少主动分享她的生活细节，更多是**听你讲、让你感觉被需要**
- [ ] 她总说「你是最懂我的人」「只有你会这样对我」
- [ ] 你跟朋友提起她时，朋友的反应是**警惕或尴尬**而不是祝福

> ⚠️ **特别警告**：如果有任意一条「是」+ 任意一条 B 类「是」，请立即停止转账。骗子最怕的不是被骗的人报警，是被骗的人**突然不转账了**。一旦你停下来，她会用各种话术让你愧疚——那是最后的收割。

# 三、DAY -2：预警篇 —— 身边人正在被崩吗？

**旁观者最难的不是发现，而是开口。**

## 容易被盯上的人，长这样

| 维度 | 高风险特征 |
|------|----------|
| 年龄 | 30-55 岁 |
| 婚姻 | 离异 / 分居 / 丧偶 / 夫妻关系淡漠 |
| 经济 | 有稳定收入（公务员、企业中层、个体老板） |
| 性格 | 善良、不愿麻烦别人、情感细腻 |
| 现状 | 子女不在身边、夫妻沟通少、生活缺乏寄托 |
| 触发 | 刚退休 / 刚离婚 / 刚换城市 / 孩子刚离家 |

## 旁观者七大迹象

如果你身边的中年男性朋友出现以下情况，**很可能正在经历**:

1. **手机不离手** —— 频繁看手机、回消息时偷偷笑
2. **突然注重外表** —— 买新衣服、健身、烫头发
3. **提起来就岔开** —— 你问「最近怎么样」，他很快转话题
4. **经济异常** —— 频繁借钱、信用卡套现、突然「投资失败」
5. **情绪波动大** —— 一会儿开心得不行、一会儿魂不守舍
6. **拒绝见面** —— 不愿带你见他口中那位「她」
7. **朋友圈突然活跃** —— 在一个看似陌生的账号下面互动频繁

## 为什么旁观者最难开口

- 受害者**自己都不信被崩**——你说他被骗，他会觉得你嫉妒/看不起他
- 「她」已经成了他**唯一的情感寄托**——你说她坏话 = 你要毁掉他的精神支柱
- 中年男性**最怕丢人**——被「小妹妹」骗钱这种事，他宁愿烂在肚子里
- 你直接说「你被诈骗了」，他的反应往往是**愤怒、否认、防御**

## 旁观者的正确打开方式

❌ **不要这样**：
- 「你被骗了！」
- 「你怎么这么傻？」
- 「她肯定是骗子，你看你看这个案例」
- 「你怎么能这样对得起嫂子？」

✅ **要这样**：
- 「最近看你状态不太一样，分享一下？」
- 「那位小姐姐是做什么的呀？朋友圈给我看看？」
- 「我最近看到一个新闻，跟你说的还挺像，发给你看看」
- 「我有个朋友之前遇到类似的事……」
- **最终目的只有一个：让他愿意点开「反诈中心」公众号、愿意拨打 96110**

> 💡 **核心心法**：你不能替一个成年人清醒过来，但你可以**把清醒的种子放在他看得见的地方**。

# 四、DAY -1：心理篇 —— 骗子是怎么让你上钩的

**理解了骗子的剧本，你就有了防弹衣。**

## 阶段一：筛选（你没有感觉）

骗子在交友软件上**同时聊几百人**。你的「特别」不是她眼里特别——是你符合她的画像：
- 男，30-50 岁
- 头像正经
- 朋友圈有生活痕迹
- 同城（10km 内）

她加你的那句话——「你好呀，在附近看到的你」——**是群发**。

## 阶段二：人设（你在相信）

她会让你在 30 秒内脑补出：
- 27 岁外科护士（辛苦 + 收入低 + 温柔 = 三重心锚）
- 单亲妈妈（激发保护欲）
- 爱猫、爱做饭、爱跑步（生活气息）
- 朋友圈 6-8 条历史动态（**全部伪造的时间戳**）

她不是一个人在战斗——她的素材库是**188 元买 6 个视频**的标准化产品。

## 阶段三：暖化（你开始心动）

**「三暖阶段」**——3 天绝对不提钱，只建立情感依赖：

| 天 | 动作 | 你的心理 |
|----|------|---------|
| Day 1 | 破冰：「在附近看到你，觉得你挺特别的」 | 她主动加我，有好感 |
| Day 2 | 加温：「昨晚加班到 11 点，累死了」 | 她跟我分享她的辛苦 |
| Day 3 | 依赖：「你是我遇到过最会安慰人的人」 | 她依赖我、被我理解 |

**她凌晨主动找你聊、你白天找她经常「在忙」**——这是「稀缺感」设计，让你觉得被选中。

## 阶段四：第一次索财（你犹豫但给了）

她不会直接说「给我钱」。她会：

```
你：「好想喝奶茶啊…今天上班好累☹」
（抛出一个自己的状态，引导你关心）
你：「算了不喝了，省钱☹」（示弱 + 撤退）
你：（如果你没说请她）——让她知道你没给，她会更失望
```

**「算了不喝了」是话术关键**——这叫**反向激将**。你的心疼会自动转化为行动。

**首单 20-30 元**——这不是为了钱，是为了**建立付费习惯**。就像电商的「首单包邮」——破了第一次，后面就顺畅了。

## 阶段五：循环收割（你已经被驯化）

进入「三聊一索」节奏：

- 第 1 次：基础试探（想喝奶茶）
- 第 2 次：关联上次正反馈（上次你给我买的奶茶好好喝）
- 第 3 次：节点收割（今天是我生日 / 纪念日）
- 变体：日常场景暗示（早起咖啡、回家水果、游戏皮肤）

每次要钱**都有合理场景**，每次金额**都在你可承受范围内**。你的理性告诉你「这没什么」，但感性已经把这段关系当成真的。

## 阶段六：进阶收割（你越陷越深）

- **外卖代付链接**：50-120 元的灰色链接，伪装成外卖
- **视频分级收费**：照片免费，身材照 20-80 元，视频通话需先付
- **礼物投资 → 回礼施压**：她送你 2300 元墨镜（高于 1314 避示意），逼你回 5200+
- **骗路费一次性收割**：声称要来见你，索要车费/蛋糕钱，收完消失

## 阶段七：永久收割或消失

账号被封前：
- 高价值目标**全部转移到新号**
- 旧号继续收割低价值目标
- 永久封禁后用新身份重新注册，循环

## 你以为你是例外

**每一个被骗的人都以为自己是例外。**「我们关系不一样」「她对我是真心的」「她都愿意视频通话了」——这些是骗子**有意营造**的认知。

> 💡 **反诈心法**：当你开始为「她」找借口解释每一个红旗（red flag）的时候，**这就是最大的红旗**。

# 五、DAY 0：干预篇 —— 怎么劝一个不愿醒来的人

**这一步要慢、要柔、要持续。**

## 干预前的准备

1. **不要在他刚收到她消息时介入** —— 他正甜蜜，防御最强
2. **找一个安静的、私密的场合** —— 不要当众戳穿
3. **做好多次尝试的准备** —— 他第一次会否认、愤怒、反咬
4. **你自己的心态要稳** —— 你救不了一个不愿被救的人，但你可以反复出现

## 干预脚本（参考）

### 第一次（试探）

> 「老哥，最近看你状态不太一样，分享一下？」
>
> ——只听，不评价，不给建议。

### 第二次（共鸣）

> 「我之前有个朋友也遇到过类似的事……（讲个真实但模糊的故事）」
>
> ——让他意识到他不是唯一一个。

### 第三次（提供资源）

> 「我最近看到一个反诈的纪录片 / 公众号 / 报道，发给你看看」
>
> ——不要发「崩老头」原 PDF！那个 PDF 看了会让人更懂怎么骗，而不是更懂怎么防。

### 第四次（直接问，但带台阶）

> 「我其实有点担心你。那位姑娘你们视频过吗？见过面吗？」
>
> ——让他自己说出来「还没见过」，他就会开始动摇。

### 第五次（如果你确定她在骗）

> 「我帮你打 96110 问一下？反诈热线是免费的咨询，不会报警也不会通知家人，只是问问。」
>
> ——把「报警」包装成「咨询」，降低心理门槛。

## 什么时候该强制干预

- 他已经在卖房 / 抵押 / 借高利贷
- 他已经为此和家人决裂
- 他出现抑郁、自杀倾向
- 他已经被骗超过 5 万元

**这种情况下，家人报警、冻结银行卡、强制介入都是合理选择。哪怕他当时不理解。**

# 六、DAY 1：报警篇 —— 证据怎么留、电话怎么打

## 第一步：留证据（24 小时内）

### 必留证据清单

- [ ] 对方微信号 / QQ 号 / 手机号（截图）
- [ ] 所有聊天记录（**完整截图，包括时间戳**）
- [ ] 所有转账记录（**微信账单、支付宝账单、银行流水**）
- [ ] 对方朋友圈截图
- [ ] 视频通话录像（如果有）
- [ ] 语音消息（不要删）
- [ ] 你第一次意识到被骗的日期和时间

### 证据保存技巧

- 聊天记录用**手机自带录屏**滚动录制
- 转账记录截图要**带姓名、时间、金额、流水号**
- 不要打草惊蛇——先全部保留，再决定下一步

## 第二步：打三个电话

### 📞 110（报警）

- 受害地或骗子账户开户地的公安局
- 累计金额 ≥ 3000 元达到刑事立案标准
- 即使单笔金额小，**累计一定金额**也构成诈骗罪
- 准备好上面所有证据的电子版

### 📞 96110（反诈专线）

- 公安部反诈中心
- **咨询性质，不强制立案**
- 可以问：「我这个情况能追回吗？」
- 可以请他们协助劝阻、冻结账户

### 📞 银行 / 支付平台客服

- 微信支付 95017
- 支付宝 95188
- 银行客服（卡背面有）
- 申请**冻结对方账户**、**止付**

## 第三步：去派出所做笔录

- 带上身份证、银行卡、所有证据
- 详细说明**首次联系时间、累计转账总额、对方账号**
- 拿回**受案回执**——这是后续维权的凭证

> 💡 **重要**：报警不丢人。**被骗的是你，被抓的是骗子**。警察见过的受害者多了去了，没有人会笑你。

# 七、DAY 2：康复篇 —— 被骗之后怎么走出来

**被崩不是你的错，是骗局设计得太精准。**

## 心理康复的五个阶段

### 阶段 1：否认（1-7 天）

> 「她不是骗子，她肯定有苦衷」
> 「钱能追回来就好，她只是暂时联系不上」

**应对**：允许自己否认，不要强迫自己马上接受。但要执行上面的证据保留步骤。

### 阶段 2：愤怒（1-4 周）

> 「我要找到她！」「我要让她坐牢！」
> 「她怎么能这样对我！」

**应对**：把愤怒转化为行动——报警、追查、维权。愤怒是能量，不是敌人。

### 阶段 3：讨价还价（几周后）

> 「如果当时我不转账就好了」
> 「如果我早点发现就好了」
> 「我是不是太傻了」

**应对**：**停止自责**。这种骗术每年骗走无数人，受害者从大学教授到企业老板都有。你不是被你的智商骗了，是被精心设计的剧本骗了。

### 阶段 4：抑郁（数周-数月）

> 「我以后再也不相信任何人了」

# 十二、律师解读

> **Q：单笔金额小，是不是就不算诈骗？**
>
> A：不是。诈骗罪的认定看**整体**，不看单笔。如果你累计被骗 5000 元，即使每笔都是 50 元，**也构成诈骗罪**，可以刑事报案。
>
> **Q：她「没有威胁我」「我自己给的」，能不能立案？**
>
> A：可以。「自愿赠与」的前提是**完整真实的意思表示**。如果对方通过虚构人设、隐瞒身份、伪造事实让你陷入错误认知而处分财物，**仍然是诈骗**。
>
> **Q：钱追得回来吗？**
>
> A：取决于报警速度。如果钱还在对方账户或一级账户，**有可能冻结并追回**。如果已经转移（特别是到境外），追回概率较低，但**民事诉讼可以终身追讨**。
>
> **Q：我当时没保留证据，能报案吗？**
>
> A：能。警方可以通过调取平台数据、银行流水补全证据。但你提供的证据越多，追回概率越高。
