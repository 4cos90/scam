# Prompt: 骗局机制科普（"我想了解崩老头是怎么运作的"）

## 使用场景

当用户想了解「崩老头」骗局的运作机制——用于学习、研究、写作、反诈科普——时使用本 prompt。

## ⚠️ 输出红线

- **只描述模式**，不复述 PDF 中的具体话术原文
- **不教学**任何操作步骤（账号注册、变声器使用等）
- **不展示**可复制的对话脚本
- **强调**心理机制和系统性问题，而非个人道德问题

---

## 执行步骤

### 第 1 步：开篇框架

用一段开场建立整体认知：

> 「崩老头不是孤立的骗术。它是一套**精心设计的、流水线化的、产业化的情感操控系统**。
>
> 理解它为什么有效，比记住它的具体话术更重要——因为骗子会更新话术，但人性不会变。
>
> 下面我从七个阶段解释它的运作机制。」

### 第 2 步：七个阶段

#### 阶段一：筛选（受害者没有感觉）

**骗子的动作**：
- 在多个交友 APP 上**同时操作几百个账号**
- 用统一标准筛选目标：男、30-50、头像正经、同城、朋友圈有生活痕迹
- 用「你好，在附近看到的你」**群发**加好友请求

**为什么有效**：
- 受害者感觉被选中（「她主动加的我」），实际上只是符合画像
- 加好友的理由很自然，没有突兀感

#### 阶段二：人设（受害者开始相信）

**骗子的动作**：
- 用统一的话术打造人设：27 岁、外科护士、单亲、爱猫爱做饭
- 朋友圈发 6-8 条伪造时间戳的动态，覆盖过去几个月
- 用 AI 生成不露全脸的照片，避免反向搜图
- 素材库是**标准化产品**——188 元买 6 个视频，可在多个账号间复用

**为什么有效**：
- 朋友圈 30 秒内让陌生人脑补出「真实、孤独、需要关心的女生」
- 时间跨度合理，让人觉得账号是「真实用了一段时间的」
- 不露全脸避免破绽，但又给受害者留下想象空间

#### 阶段三：暖化（受害者开始心动）

**三个子阶段，每个子阶段有明确的目标和指标**：

| 子阶段 | 目标 | 持续时间 | 关键动作 |
|--------|------|---------|---------|
| 破冰 | 让对方愿意持续聊天 | Day 1 | 「在附近看到你，觉得你挺特别的」 |
| 加温 | 让对方开始关心自己 | Day 2 | 「昨晚加班到 11 点，累死了」+ 引导自我介绍 |
| 建立依赖 | 让对方觉得自己被需要 | Day 3 | 制造脆弱面 + 「你是我遇到过最会安慰人的人」 |

**为什么有效（心理学机制）**：

| 机制 | 解释 |
|------|------|
| **稀缺感** | 她凌晨主动找你、白天「在忙」，让你觉得被选中 |
| **身份锚定** | 「你是最懂我的人」——给你贴标签，让你舍不得离开 |
| **情感投射** | 她只展示你需要看到的部分，让你填补剩余部分 |

**严格纪律**：这 3 天**绝对不提钱**。目标是建立情感依赖。

#### 阶段四：第一次索财（受害者犹豫但给了）

**关键设计**：

- **首次金额极小**（20-50 元）—— 不是为钱，是为建立付费习惯
- **场景化借口** ——「好想喝奶茶」「加班错过了末班车」「昨晚聊太晚起不来」
- **反向激将** ——「算了不喝了」—— 让受害者主动付款

**心理学原理：登门槛效应（Foot-in-the-door）**

> 一旦受害者支付了第一笔，第二次、第三次、第四次就越来越容易。每一笔都比上一笔稍大，但仍在他可承受范围内。

**正反馈闭环**：

```
受害者转账 → 受害者立刻收到夸张感激（"啊啊啊啊谢谢哥哥"）
       ↓
受害者感觉良好（"20 块钱买到一个女生的感激和依赖，太值了"）
       ↓
下一笔更容易
```

**这一步的目的不是收益，是建立付费习惯**——就像电商的「首单包邮」。

#### 阶段五：循环收割（受害者已经被驯化）

**节奏：三聊一索**

> 聊 3 天 → 索 1 次 → 再聊 3 天 → 再索 1 次

**话术升级路径**：

| 轮次 | 策略 | 心理学原理 |
|------|------|----------|
| 第 1 次 | 基础试探 | 直接场景化（奶茶、打车） |
| 第 2 次 | 关联上次正反馈 | 沉没成本 + 锚定效应（"上次你给我买的奶茶好好喝"） |
| 第 3 次 | 节点收割 | 权威借势（"今天是我生日"） |
| 变体 | 日常场景暗示 | 多样性避免警觉 |

**关键设计**：
- 每次索财都有**具体的、可信的场景**
- 每次金额都在受害者可承受范围内
- 受害者理性告诉自己「这没什么」，但感性已经把关系当真

#### 阶段六：进阶收割（受害者越陷越深）

当受害者已经为这个人转过多次账后，进入进阶手段：

| 进阶手段 | 原理 | 预期收益 |
|---------|------|---------|
| 外卖代付链接 | 制造「为你着想」的假象 | 50-120 元 / 次 |
| 视频分级收费 | 制造稀缺感 | 20-100 元 / 次 |
| 礼物投资 → 回礼施压 | 互惠原理 + 锚定效应 | 5200+ 元 |
| 骗路费一次性收割 | 见面的承诺 + 信任 | 数千到上万 |

**为什么受害者不警觉**：

- 每一笔金额相对总收入都不算大
- 关系已经建立信任
- 「她不会骗我」是认知失调的产物

#### 阶段七：永久收割或消失

**账号生命周期管理**：

- Week 1-4：活跃期——平台信用高、推荐流量多
- Week 5-12：稳定期——账号热度下降，开始出现平台警告
- Week 13-26：衰退期——可能被多次举报

**关键操作**：
- 高价值目标**全部转移到新号**
- 旧号继续收割低价值目标
- 永久封禁后用新身份重新注册，循环

**这就是为什么「崩老头」灰产能持续运转**。

### 第 3 步：心理学原理总结

用一个表格总结所有心理操控机制：

| 原理 | 描述 | 在崩老头中的应用 |
|------|------|----------------|
| **稀缺感** | 觉得得不到的东西更珍贵 | 主动深夜找你，但白天经常「在忙」 |
| **登门槛效应** | 一旦答应小要求，更容易答应大要求 | 20 元 → 50 元 → 100 元 → 500 元 |
| **反向激将** | 用「算了」激发对方的主动付出 | 「算了不喝了」「算了太贵了换个便宜的」 |
| **沉没成本** | 已经投入越多，越不愿意放手 | 已经投入时间、金钱、感情 |
| **锚定效应** | 受初始数字影响后续判断 | 2300 元墨镜锚定回礼金额 |
| **间歇性强化** | 不定期给奖励比固定奖励更有效 | 不是每次要钱都给，但有时候会让受害者觉得自己被偏爱 |
| **认知失调** | 行为与信念矛盾时，人们倾向于改变信念 | 已经投入这么多，「她肯定是真的」 |
| **身份锚定** | 给对方贴标签，强化承诺 | 「你是最懂我的人」「你是第一个给我买 X 的人」 |
| **互惠原理** | 收到好处后倾向于回报 | 寄 2300 元礼物让受害者回礼 |
| **权威借势** | 利用社会身份增加可信度 | 「医院外科护士」= 辛苦 + 低收入 + 温柔 |
| **稀缺 + 排他** | 「只对你这样」增加独占感 | 「你是我遇到过最……」 |

### 第 4 步：产业链结构

解释为什么这是**产业化**而非个体行为：

```
上游：培训师
  - 卖教程（5-9.9 元 / 份）
  - 卖素材包（188 元 / 6 视频）
  - 卖变声器（几十元 - 800 元）
  - 标准化话术脚本
  - 防举报指南

中游：执行者
  - 批量操作
  - 一人维护 30-300 个目标
  - 日流水可达 800+ 元
  - 月收入 1-3 万元

下游：代崩 / 回收团队
  - 接手「养肥」账号
  - 二次收割
  - 资金转移
```

**典型案例（已破获）**：

山东淄博警方破获的团伙——14 名男性伪装女性——流水化分工：
- 50+ 部手机
- 20+ 台电脑
- 20+ 个变声器
- 引流组 → 日常聊天组 → 索财组 → 资金转移组
- **80 余名受害人，涉案 100 余万元**

### 第 5 步：为什么受害者极少报警

| 原因 | 解释 |
|------|------|
| 单笔金额太小 | 几十到几百，不够立案心理感知 |
| 怕丢人 | 已婚男性被「小妹妹」骗钱，暴露后家庭风险更大 |
| 难以定性 | 「自愿发的红包」 vs 「被诈骗」界限模糊 |
| 认知失调 | 已经投入太多，不愿承认被骗 |
| 仍抱希望 | 「她可能真的有苦衷」「这次之后再也不给了」 |
| 实际案例 | 有受害者两年内累计损失超 8 万元却未维权 |

**这是灰产能持续运转的根本原因。**

### 第 6 步：法律边界

| 情形 | 法律定性 |
|------|---------|
| 单次自愿赠与 | 民事纠纷，难以定性 |
| 虚构事实索财 | 涉嫌诈骗罪 |
| 累计 ≥ 3000 元 | 达到诈骗罪刑事立案标准 |
| 累计 ≥ 50 万元 | 十年以上有期徒刑 |
| 多人团伙 | 共同犯罪，主犯从重 |

### 第 7 步：法律边界与维权要点

如果你或你身边人正在经历：

1. **立即保留证据** —— 截图、账单、对方账号
2. **拨打 96110 反诈专线** —— 咨询 + 协助冻结
3. **累计 ≥ 3000 元拨打 110 报警** —— 无论单笔多小
4. **民事诉讼可终身追讨** —— 即使对方跑到境外

### 第 8 步：让用户带走的最重要的认知

用一段话总结：

> 「崩老头之所以能运转，不是技术多高明，而是因为它**把每一笔都设计成'不像诈骗'**。
>
> 20 块、30 块、52 块——看起来都是'自愿的小礼物'。受害者每一次转账都感觉'这没什么'。但等到反应过来，账户已经被掏空了。
>
> 这种骗局利用的不是智商，而是**人性**——善良、孤独、对情感的渴望、对'被年轻异性关注'的隐性渴望。
>
> 每一个受害者都以为是例外。但**正是这种'我不会是例外'的信念，让骗子一次次得逞**。」
